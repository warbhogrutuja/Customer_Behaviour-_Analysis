# 📊 Data Analytics Project

## Overview

This project demonstrates a complete Data Analytics workflow, covering data extraction, cleaning, analysis, visualization, and reporting. The goal is to transform raw data into meaningful insights that support data-driven decision-making.

The project includes:

- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using PostgreSQL, MySQL, and SQL Server
- Interactive dashboard creation in Power BI
- Business report generation
- Presentation creation using Gamma

## How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### Navigate to Project Directory

```bash
cd your-repository-name
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Analysis

1. Open Jupyter Notebook.
2. Run the notebooks in the following order:
   - Data Loading
   - EDA
   - Data Cleaning
3. Export cleaned data.
4. Import data into PostgreSQL/MySQL/SQL Server.
5. Execute SQL scripts.
6. Open the Power BI dashboard file.
7. Review the report and presentation.
